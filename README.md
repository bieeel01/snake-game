# 🐍 Snake Game

Um jogo clássico da cobrinha desenvolvido com **HTML5, CSS3 e JavaScript Vanilla**, executado diretamente no navegador sem dependências externas.

---

## 🎮 Demonstração

Controle a cobra, colete alimentos, aumente sua pontuação e tente superar seu recorde sem colidir com as paredes ou com o próprio corpo.

---

## ✨ Funcionalidades

- 🐍 Jogabilidade clássica do Snake
- 🍎 Sistema de geração aleatória de comida
- 📈 Contador de pontuação em tempo real
- 🏆 Sistema de recorde utilizando Local Storage
- 💀 Tela de Game Over
- 🔄 Reinício rápido da partida
- 🎨 Interface moderna com visual neon
- ⚡ Executado inteiramente no navegador

---

## 🎯 Como Jogar

### Controles

| Tecla | Ação |
|--------|--------|
| ⬆️ | Mover para cima |
| ⬇️ | Mover para baixo |
| ⬅️ | Mover para esquerda |
| ➡️ | Mover para direita |

### Objetivo

1. 🍎 Coma os alimentos espalhados pelo mapa.
2. 📏 Aumente o tamanho da cobra.
3. 🏆 Alcance a maior pontuação possível.
4. 🚫 Evite bater nas paredes.
5. 🚫 Evite colidir com o próprio corpo.

---

## ⚙️ Mecânicas do Jogo

### Sistema de Pontuação

- Cada alimento coletado adiciona **1 ponto**.
- O recorde é salvo automaticamente no navegador.

### Sistema de Colisão

O jogo termina quando:

- A cobra atinge uma parede.
- A cobra colide com seu próprio corpo.

### Sistema de Crescimento

Ao consumir um alimento:

- A pontuação aumenta.
- A cobra cresce.
- Um novo alimento é gerado em posição válida.
